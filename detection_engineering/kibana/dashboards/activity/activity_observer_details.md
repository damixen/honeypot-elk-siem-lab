# Activity Observer - Details
### Purpose
This dashboard is used to view activity at the source IP level. This is used to identify the dominant IP address and port access pattern. 

## Questions Answered
- What is the dominant IP?
- How many ports did the IP access?
- Where did the IP address originate?
- What is the distribution of the IP access?
