# Activity Observer Overview Dashboard

## Purpose
This dashboard is used to observe network/honeypot activity at a high level to identify activity level, activity trend, geographic information, and port/service statistics. 

### Key Visualizations
- **Network Connection Flow Count**
	- Uses Suricata's "flow" event type, which includes activities such as SYN scan, which only establishes a connection to the target port, but never proceeds to service interaction. 
- **Honeypot Activity**
	- Uses events generated by each honeypot. The honeypot events do not include network-level connection activity, unlike Suricata.
### Questions Answered
- When did network activities increase?
- When did honeypot activities increase?
- Which port had the most activities?
- Which country or ASN had the most activities?
- What's the duration of network activities?
