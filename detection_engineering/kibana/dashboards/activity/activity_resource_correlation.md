# Activity Resource Correlation Dashboard

## Purpose
This dashboard is used to observe the correlation among activity, system resource, and container resource. This view helps assess whether increased activity corresponds with elevated infrastructure utilization.

### Questions Answered
- Did the system and container resource usage increase with honeypot activity?
- Do system resource usage and container resource usage correlate?
