# SMB Command View - Short Range

## Purpose
Used to observe SMB command behavior for detection rule creation and alert trigger annotations. This dashboard is meant to be used to check a short time range where the timestamp aggregation level is smaller. By having wide range and short range, the Elasticsearch look-up time can be optimized for desired aggregation level.
