# SMB Stats

## Purpose
This dashboard shows various SMB related views, including top SMB Commands, SMB Command and Status pair, SMB Engagement funnel and timeline, and Session Setup Success rate.

## Questions Answered
- What SMB commands were observed?
- What SMB Engagement Funnel observed?
	- (SMB Engagement Funnel is defined in a case study)
- What is the ratio of observed network access and SMB sessions?
	- Fewer SMB sessions but more network access means more SYN scan type activity.
- What is the ratio of SMB Session Setup success?
	- A lower success rate may indicate more login attempts.
